Build from source.

-------- Windows
Install latest Lazarus IDE
http://www.lazarus.freepascal.org/

Open project file:
menu Project -> open project
select 'trackereditor.lpi' file inside 'source/project/tracker_editor/'

Build:
Menu Run -> Build
Executable file is created in the 'enduser' directory.

-------- Linux Ubuntu 14.04
sudo apt-get update
sudo apt-get install lazarus -y
cd source/project/tracker_editor/
lazbuild trackereditor.lpi

Executable file is created in the 'enduser' directory.
